📱 Sobre o Projeto
Aplicativo móvel desenvolvido com React Native via Expo Snack, que calcula o Índice de Massa Corporal (IMC) do usuário e exibe a classificação de acordo com a tabela da Organização Mundial da Saúde (OMS).

✅ Funcionalidades

Entrada de peso (kg) e altura (m)
Cálculo automático do IMC
Classificação com cores conforme tabela OMS
Validação de dados inseridos
Botão para limpar os campos
